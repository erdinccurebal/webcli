webcli

55+ commands to navigate, snapshot, and interact with web pages from the terminal. Built on Playwright.

$ npm install -g @erdinccurebal/webclicopy
Terminal
$ webcli go https://example.com
Navigated to https://example.com (tab: default)

$ webcli snapshot
[document]
  [heading] "Example Domain"
  [link @e1] "More information..."
(1 interactive refs)

$ webcli click @e1
Clicked "@e1" → https://www.iana.org/help/example-domains

$ webcli title
IANA — Example Domains

$ webcli screenshot --annotate -o page.png
Annotated screenshot saved: page.png (12 elements)

$ webcli state save session.json
State saved: session.json

Features

55+ Commands

Navigate, read, interact, snapshot, screenshot, PDF, console, storage, state management, device emulation, and more.

Accessibility Snapshot

Get the page's accessibility tree with deterministic refs (@e1, @e2) for reliable element targeting.

Persistent Daemon

Browser stays open between commands. No startup overhead. Auto-shuts down after idle timeout.

Multi-Tab

Manage multiple named browser tabs simultaneously with the -t flag.

Anti-Detection

Realistic user-agent, webdriver flag removal, configurable viewport. Works on bot-protected sites.

State Management

Save and restore full browser state — cookies, localStorage, and sessionStorage — as JSON.

Device Emulation

Emulate any device — iPhone, iPad, Pixel, and more — with accurate viewport and user-agent.

JSON Output

Every command supports --json flag for structured, machine-readable output.

Agent Skill

Works as a skill for Claude Code, Cursor, Copilot, Codex, Windsurf and 30+ other AI agents.
