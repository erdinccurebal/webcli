browsercli

Navigate, scrape, and interact with web pages from the terminal. Built on Playwright.

$ npm install -g browsercli click to copy
Terminal
$ browsercli go https://example.com
Navigated to https://example.com (tab: default)

$ browsercli source
[Page: https://example.com]
Example Domain
This domain is for use in illustrative examples...

$ browsercli click "More information..."
Clicked "More information..." → https://www.iana.org/help/example-domains

$ browsercli screenshot -o page.png
Screenshot saved: page.png

$ browsercli stop
Daemon stopped

Features

30+ Commands

Navigate, read content, fill forms, click buttons, take screenshots, manage cookies, and more.

Persistent Daemon

Browser stays open between commands. No startup overhead. Auto-shuts down after idle timeout.

Multi-Tab

Manage multiple named browser tabs simultaneously with the -t flag.

Anti-Detection

Realistic user-agent, webdriver flag removal, configurable viewport. Works on bot-protected sites.

Session Persistence

Cookies and localStorage persist across restarts with configurable user data directory.

JSON Output

Every command supports --json flag for structured, machine-readable output.

Commands

Navigation

  • go Navigate to URL
  • back Go back
  • forward Go forward
  • reload Reload page

Reading

  • source Page text content
  • html Element innerHTML
  • attr Element attribute
  • links List all links
  • forms List all forms
  • eval Run JavaScript

Interaction

  • click Click by text
  • clicksel Click by selector
  • fill Fill input field
  • select Select dropdown
  • type Type with keyboard
  • press Press key
  • focus Focus element

Waiting

  • wait Wait for selector
  • waitfor Wait for text
  • sleep Sleep N ms

Browser

  • screenshot Capture page
  • cookie Export/import cookies
  • viewport Change viewport
  • useragent Change UA
  • network Request logging

Management

  • tabs List open tabs
  • quit Close a tab
  • status Daemon info
  • stop Stop daemon
